# joonggo-market-service

### 개발동기

- 중고 거래 서비스의 기능을 직접 구현하여 백엔드 시스템에 대한 이해도를 높이고 싶었습니다.

### 사용 기술 - 구현
- Spring 5
- Maven 3.5.2
- JDK 1.8
- MySQL 8.0.27
- Redis 2.1.3

### 사용 기술 - 배포
- AWS(EC2/RDS/S3)
- Jenkins
- GitHub
