# joonggo-market-service

### 개발동기

- 중거 거래 서비스의 기능들을 직접 구현하여 백엔드 시스템에 대한 이해도를 높이고 싶었다.
- 중고 거래 플랫폼 [번개장터](https://m.bunjang.co.kr/)를 참고했다.

### 사용기술

- Spring 5
- JDK 1.8
- MySQL
- AWS
